# Pokédex Final

Uma aplicação desenvolvida em React que consome dados da PokéAPI para exibir informações detalhadas de Pokémon, incluindo tipos, estatísticas, habilidades, peso, altura e movimentos.

## Funcionalidades

* Listagem dos primeiros 20 Pokémon
* Visualização detalhada de cada Pokémon
* Exibição de tipos com cores personalizadas
* Estatísticas base com barras de progresso
* Informações de espécie, altura, peso e habilidades
* Interface responsiva para desktop e dispositivos móveis

## Tecnologias Utilizadas

* React
* JavaScript (ES6+)
* Vite
* PokéAPI

## Como Executar o Projeto

### Pré-requisitos

* Node.js instalado (versão 18 ou superior recomendada)
* npm instalado

### 1. Clone o repositório

```bash
git clone https://github.com/pedroaborges/pokedex-final.git
```

### 2. Acesse a pasta do projeto

```bash
cd pokedex-final
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute a aplicação

```bash
npm run dev
```

### 5. Acesse no navegador

Após iniciar o servidor, abra o endereço exibido no terminal, normalmente:

```text
http://localhost:5173
```

## Fonte dos Dados

Os dados dos Pokémon são obtidos através da API pública:

* PokéAPI: https://pokeapi.co/
